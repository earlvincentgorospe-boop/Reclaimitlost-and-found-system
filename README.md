a basic lost and found system 
Backend language: PHP (procedural PHP pages and API handlers), index.php, config.php, ajax.php.
Frontend languages: HTML, CSS, JavaScript, index.php, style.css, main.js:1.
Database: MySQL (using PHP mysqli), configured in config.php:1 and created by setup_database.php:1.
How to Run

Install XAMPP, then start Apache and MySQL.
Put this folder inside C:/xampp/htdocs/ (keep the folder name as is).
Open: http://localhost/reclaimit-lost%20and%20found%20system/setup_database.php to auto-create DB/tables.
Open: http://localhost/reclaimit-lost%20and%20found%20system/ to use the app.
